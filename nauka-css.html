tydzień 2, lekcja 3

stylowanie po selektorze - podstawy

stylujemy po klasie; klasa to atrybut znacznika HTML, który zapisujemy class="name"
aby ostylować klasę w CSS odwołujemy się do niej w następujący sposób: 

                
                    .name {
                        [tu podajemy właściwości, które chcemy ostylować]
                    }

będą one miały zastosowanie do każdego znacznika, któremu nadamy klasę name

możliwe jest też stylowanie po id, ale jest to zła praktyka, wówczas w CSS do id odwołujemy się w następujący sposób: 

                
                    #name {
                        [tu określamy właściwości, które będą miały zastosowanie do znacznika, któremu nadamy to id]
                    }

specyficzność w CSS

  1. !important
  2. style=" " - tzw. znacznik inline
  3. #id - stylowanie po id
  4. .class - stylowanie po klasie
  5. h1 - stylowanie po elemncie

specyficzność to coś w rodzaju klasyfikacji medalowej: drużyna, która zdobyła 1 złoty medal jest wyżej niż tak, która zdobyła 10 srebrnych

np.: 

                    
                        body h1 {
                            color: yellow;
                        }

                        h1 {
                            color: red;
                        }

w tej sytuacji stylowany element będzie żółty, bo jednocześnie stylujemy dwa elementy

Podsumowanie:

  • używaj pojedynczych słów z języka angielskiego do nazywania klas
  • nazywaj rzeczy zgodnie z ich funkcją na stronie
  • startuj z małej litery
  • nie zaczynaj od cyfry
  • jeśli musisz użyć dwóch lub więcej słów, to używaj kebab-case
  • nie zwiększaj specyficzności bez potrzeby, staraj się, żeby element miał jedną klasę
