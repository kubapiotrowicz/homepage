Tu są moje notatki z nauki JavaScript

tydzień 3, lekcja 7

JavaScript != Java
kod JS uruchamia się tylko raz (po załadowaniu strony) JavaScript składa się z:

  • wartości
  • operacje porównywania
  • literały
  • zmienne
  • obiekty
  • funkcje

wartości są dla JS tym, co liczby dla matematyki - one już istnieją we wszechświecie wartości
pliki zawierające kod JS mają rozszerzenie .js i mają swój znacznik script - dodajemy go na samym dole, przed końcem body

w JS korzystamy z mnóstwa gotowych rzeczy, np. console.log()

zmienna w JS jest połączeniem między jej nazwą a wartością, do której się odnosi; korzystamy z niej po to, żeby łatwiej i szybciej dało się odwoływać do wartości, do której się odwołuje

w przypadku nazw zawierających dwa lub więcej wyrazów możęmy korzystać z camelCase

backtick ` - wykorzystywany w literałach, np.: `Jestem ${name}.`
